IOT BASED SMART DUSTBIN MANAGEMENT SYSTEM 1

IoT-Based Smart Dustbin Management System Project Overview This project involves creating a smart dustbin management system using Arduino, a GSM module, and an ultrasonic sensor. The system monitors the waste level in the dustbin and sends an alert when it reaches a predefined threshold, improving waste management efficiency. Components Used Arduino Uno (or any compatible Arduino board) Ultrasonic Distance Sensor (HC-SR04) GSM Module (SIM800L or SIM900) Breadboard and Jumper Wires Power Supply (Battery or USB) Basic Electronic Components (Resistors, etc.) Optional: LEDs or Buzzers for additional status indication Project Description The smart dustbin management system uses an ultrasonic sensor to measure the waste level inside the dustbin. When the waste level reaches a critical point, the system sends an SMS notification via the GSM module to the waste collection service, informing them to collect the waste. System Components and Connections 1. Hardware Setup Ultrasonic Sensor (HC-SR04) VCC to 5V on Arduino GND to GND on Arduino Trig to Digital Pin 9 Echo to Digital Pin 10 GSM Module VCC to a suitable power supply (e.g., 5V or 12V based on module specifications) GND to GND on Arduino TX to Arduino RX (Pin 2) RX to Arduino TX (Pin 3) 2. Arduino Code cpp #include const int trigPin = 9; const int echoPin = 10; const int thresholdDistance = 20; // Alert threshold in cm SoftwareSerial gsmSerial(2, 3); // RX, TX void setup() { pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); gsmSerial.begin(9600); // GSM module baud rate Serial.begin(9600); // Serial Monitor baud rate sendSMS("Initialization complete. Dustbin monitoring system is active."); } void loop() { long duration; int distance; digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); duration = pulseIn(echoPin, HIGH); distance = duration * 0.0344 / 2; Serial.print("Distance: "); Serial.println(distance); if (distance < thresholdDistance) { sendSMS("Alert: Dustbin is full or nearly full."); delay(60000); // Prevent multiple alerts in quick succession } delay(1000); // Measurement interval } void sendSMS(String message) { gsmSerial.println("AT"); delay(1000); gsmSerial.println("AT+CMGF=1"); // Set SMS mode to text delay(1000); gsmSerial.println("AT+CMGS=\"+1234567890\""); // Recipient's phone number delay(1000); gsmSerial.print(message); delay(100); gsmSerial.write(26); // Send SMS delay(5000); } Testing and Deployment Upload the Code: Transfer the code to the Arduino using the Arduino IDE. Power Up: Ensure correct power supply for both Arduino and GSM module. Monitor Output: Use the Serial Monitor to observe distance readings and debug information. Simulate Full Dustbin: Place an object near the ultrasonic sensor to test SMS alert functionality. Deployment Considerations: Enclosure: Secure the Arduino and modules in a weather-resistant enclosure for outdoor use. Power Supply: Consider using a battery or solar panel for uninterrupted operation. Future Enhancements Real-time Data: Integrate with cloud services for real-time monitoring and data analytics. Web Interface: Develop a web application for managing multiple dustbins and viewing status updates.
